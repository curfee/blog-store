如果要使用dplayer需选装插件
```
npm install hexo-tag-dplayer --save
```
然后就可以自定义视频了
```
{%
    dplayer
    "url=2019minecraft.mp4"
    "loop=yes"
    "theme=#FADFA3"
    "autoplay=false"
    "token=tokendemo"

%}
```
